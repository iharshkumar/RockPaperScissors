# Rock Paper Scissors

A simple browser-based Rock Paper Scissors game built with HTML, CSS, and JavaScript.

## How to Use

1. Open `index.html` in your browser.
2. Click one of the three choices: rock, paper, or scissors.
3. The computer picks a random move.
4. The score updates and a message shows the result.

## Files

- `index.html` - page structure and game layout
- `style.css` - styling for the game interface
- `app.js` - game logic and score handling
- `images/` - icons for rock, paper, and scissors

## Notes

- This is a client-side app and does not require a server.
- The game tracks the user score and computer score during the session.
